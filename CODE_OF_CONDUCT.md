# Code of Conduct

This project aims to be a welcoming, respectful, and professional space for
everyone interested in learning and sharing Computer Science resources.

---

## Our Standards

I expect all participants to:
- be respectful and constructive
- welcome different viewpoints and ideas
- communicate clearly and professionally

Unacceptable behavior such as harassment or hostility is not welcome here.

---

## Moderation

I may take appropriate action if behavior does not align with these
guidelines, in order to keep the project healthy and collaborative.

---

## Scope

These guidelines apply to all project spaces, including:
- issues
- pull requests
- discussions
